# Release v2.0.0 — 大更新

发行说明（概要）
- 版本：v2.0.0
- 主要更新：添加收藏题目功能、支持从 Word 导入、提高题目解析识别范围

新功能
1. 添加“收藏题目”功能
   - 用户可以将任意题目标记为收藏，并在专用收藏列表中查看/管理收藏题目。
2. 支持从 Word 文档（.docx）导入题目
   - 支持批量导入题目，自动解析题目内容、选项、题型和答案。
   - 建议用于标准化/结构化的 .docx；对非结构化格式提供较好的容错能力。
3. 提高解析识别范围
   - 题目内容、题目类型和题目答案的识别不再仅限于传统或固定标记（如 "A/O"），增强对多样化题目格式的支持。
   - 解析逻辑已重构以便后续扩展更多识别规则与格式。

优化与兼容性提示
- 导入过程增强了错误提示与容错处理，减少人工干预。
- 若此前项目依赖固定标记解析（如只识别 “AO”），请在导入后在测试环境核对结果，必要时调整解析规则或样例文档格式。
- 本次为向后兼容的功能增强，版本号从 1.x 跳至 2.0.0 以示重大功能集成（非破坏性 API 改动仍需按需确认）。

开发/部署说明（可选）
- 推荐使用 python-docx 等库解析 .docx（如需，我可以列出实现建议或样例代码）。
- 建议在测试分支使用若干样例 .docx 验证导入与解析效果，再在生产环境批量导入。
